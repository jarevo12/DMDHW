Weekly Goals Dashboard Concepts

Three options for showing weekly-goal habit evolution across weeks. All use the same sample data for comparison.

Option A — Heatmap Grid (Habits x Weeks)

Fast pattern recognition. Good when there are many weekly-goal habits.

0% 50% 100%

Option B — Small Multiples (Mini Bars per Habit)

Clear per-habit progress trend while keeping the list compact.

Option C — Weekly Cards (Each Week Shows All Habits)

Best for week-by-week review and coaching. Scales well up to ~5 habits.
